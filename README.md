#  Expense Management System

A web-based application to track and manage daily expenses built with PHP and MySQL.

##  Features

-  User Login & Registration
-  Add / Edit / Delete Expenses
-  View Today, Yesterday, Last Week & Last Month Expenses
-  Search Expenses by Date Range
-  Admin Panel to Manage Users
-  User Profile Picture Upload

##  Technologies Used

- HTML, CSS, Bootstrap 5
- JavaScript
- PHP
- MySQL

##  Installation

1. Clone the repo
2. Import `database.sql` into phpMyAdmin
3. Configure `includes/db_conn.php` with your DB credentials
4. Run on XAMPP localhost

##  Database

- Database name: `expense`
- Import file: `database.sql`
